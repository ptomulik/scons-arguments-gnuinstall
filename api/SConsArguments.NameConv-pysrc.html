 31      """Provides a systematic way to transform *argument* names into keys 
 32      identifying their *endpoints*. 
 33   
 34      This object may be helpful when generating multiple *arguments* where 
 35      the *endpoint* names may have to be derived from *argument* names in 
 36      a systematic way. The object provides four lambdas, which may be used 
 37      to transform *argument* names to *endpoint* names. By default, these 
 38      lambdas prepend preset prefixes and append preset suffixes to the name. 
 39      For option keys and names, the original key is lowercased. These prefixes, 
 40      as well as lambdas may be overwritten by user. 
 41   
 42      :Ivar env_key_prefix: 
 43          a prefix that is by default prepended to ENV key by the 
 44          `name2env` lambda, default: ``''`` 
 45      :Ivar env_key_suffix: 
 46          a suffix that is by default prepended to ENV key by the 
 47          `name2env` lambda, default: ``''`` 
 48      :Ivar var_key_prefix: 
 49          a prefix that is by default prepended to VAR key by the 
 50          `name2var` lambda, default: ``''`` 
 51      :Ivar var_key_suffix: 
 52          a suffix that is by default prepended to VAR key by the 
 53          `name2var` lambda, default: ``''`` 
 54      :Ivar opt_key_prefix: 
 55          a prefix that is by default prepended to OPT key by the 
 56          `name2opt` lambda, default: ``''`` 
 57      :Ivar opt_key_suffix: 
 58          a suffix that is by default prepended to OPT key by the 
 59          `name2opt` lambda, default: ``''`` 
 60      :Ivar opt_prefix: 
 61          a prefix that is by default used when composing option names, 
 62          usually a single or double dash, default: ``'--'`` 
 63      :Ivar opt_name_prefix: 
 64          additional prefix used when composing option names, inserted 
 65          between `opt_prefix` and the *argument* name, default: ``''`` 
 66      :Ivar opt_name_suffix: 
 67          a suffix that is by default used when composing option names, 
 68          default: ``''`` 
 69

 73          import SConsArguments 
 74          tr = SConsArguments._ArgumentNameConv(env_key_prefix = 'ENV_', env_key_suffix = '_VNE',
