106      function is for internal use and IS **NOT a part of public API**. 
107   
108      :Parameters: 
109          value 
110              the value to be processed; if it is a string, it is passed through 
111              placeholder renaming procedure; otherwise it is returned unaltered, 
112          resubst_dict 
113              a dictionary of the form ``{ "xxx":"${yyy}", "vvv":"${www}", ...}`` 
114              used to rename placeholders within `value` string; with the above 
115              dictionary, all occurrences of ``$xxx`` or ``${xxx}`` within 
116              `value` string will be replaced with ``${yyy}``, all occurrences of 
117              ``$vvv`` or ``${vvv}`` with ``${www}`` and so on; see also 
118              `_build_resubst_dict()`, 
119      :Returns: 
120          returns the `value` with placeholders renamed. 
121      """ 
122      if SCons.Util.is_String(value): 
123          # make substitution in strings only 
124          return string.Template(value).safe_substitute(**resubst_dict) 
125      else:

130      """Build dictionary for later use with `_resubst()`. This function is for 
131      internal use and IS **NOT a part of public API**. 
132   
133      **Example** 
134   
135      .. python:: 
136   
137          >>> import SConsArguments 
138          >>> SConsArguments._build_resubst_dict( {"xxx":"yyy", "vvv":"www", "zzz":"zzz" }) 
139          {'vvv': '${www}', 'xxx': '${yyy}'} 
140   
141      :Parameters: 
142          rename_dict : dict 
143              dictionary of the form ``{"xxx":"yyy", "vvv":"www", ...}`` 
144              mapping variable names from one namespace to another, 
145   
146      :Returns: 
147          returns dictionary of the form ``{"xxx":"${yyy}", "vvv":"${www}"}`` 
148          created from `rename_dict`; items ``(key, val)`` with ``key==val`` are

150          result. 
151      """
