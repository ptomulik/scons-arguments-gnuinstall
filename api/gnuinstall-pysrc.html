Source Code for Module gnuinstall

  1  """`gnuinstall` 
  2   
  3  Provides `GNU installation directories`_ as SCons *arguments*. 
  4   
  5  **General Description** 
  6   
  7  This module provides standard `GNU installation directories`_ defined by `GNU 
  8  Coding Standards`_, for example ``$prefix``, ``$bindir`` or ``$sysconfdir``. 
  9  The variables defined here may be easilly added to: 
 10   
 11      - SCons environment, as construction variables (``env.subst('$variable')``), 
 12      - SCons command line variables (``scons variable=value``), 
 13      - SCons command line options (``scons --variable=value``). 
 14   
 15  **Quick start** 
 16   
 17  .. python:: 
 18      # SConstruct 
 19      import SConsArguments 
 20   
 21      env = Environment() 
 22      env.Replace(install_package = 'my_install_package', package = 'my_package') 
 23      var = Variables() 
 24      decls = SConsArguments.ImportArguments('gnuinstall') 
 25      args = decls.Commit(env, var, True) 
 26      args.Postprocess(env, var, True) 
 27   
 28      print env.subst("prefix : ${prefix}") 
 29      print env.subst("bindir : ${bindir}") 
 30   
 31  Running examples:: 
 32   
 33      ptomulik@barakus:$ scons -Q 
 34      prefix : /usr/local 
 35      bindir : /usr/local/bin 
 36   
 37      ptomulik@barakus:$ scons -Q prefix=/usr 
 38      prefix : /usr 
 39      bindir : /usr/bin 
 40   
 41   
 42  **Supported variables** 
 43   
 44    prefix 
 45        Installation prefix 
 46    exec_prefix 
 47        Installation prefix for executable files 
 48    bindir 
 49        The directory for installing executable programs that users can run. 
 50    sbindir 
 51        The directory for installing executable programs that can be run from the 
 52        shell, but are only generally useful to system administrators. 
 53    libexecdir 
 54        The directory for installing executable programs to be run by other 
 55        programs rather than by users. 
 56    datarootdir 
 57        The root of the directory tree for read-only architecture-independent 
 58        data files. 
 59    datadir 
 60        The directory for installing idiosyncratic read-only 
 61        architecture-independent data files for this program. 
 62    sysconfdir 
 63        The directory for installing read-only data files that pertain to a single 
 64        machine - that is to say, files for configuring a host. 
 65    sharedstatedir 
 66        The directory for installing architecture-independent data files which 
 67        the programs modify while they run. 
 68    localstatedir 
 69        The directory for installing data files which the programs modify while 
 70        they run, and that pertain to one specific machine. 
 71    includedir 
 72        The directory for installing header files to be included by user programs 
 73        with the C ``#include`` preprocessor directive. 
 74    oldincludedir 
 75        The directory for installing ``#include`` header files for use with compilers 
 76        other than GCC. 
 77    docdir 
 78        The directory for installing documentation files (other than Info) for this 
 79        package. 
 80    infodir 
 81        The directory for installing the Info files for this package. 
 82    htmldir 
 83        Directory for installing documentation files in the html format. 
 84    dvidir 
 85        Directory for installing documentation files in the dvi format. 
 86    pdfdir 
 87        Directory for installing documentation files in the pdf format. 
 88    psdir 
 89        Directory for installing documentation files in the ps format. 
 90    libdir 
 91        The directory for object files and libraries of object code. 
 92    lispdir 
 93        The directory for installing any Emacs Lisp files in this package. 
 94    localedir 
 95        The directory for installing locale-specific message catalogs for this 
 96        package. 
 97    mandir 
 98        The top-level directory for installing the man pages (if any) for this 
 99        package. 
100    man1dir .. man9dir 
101        Simmilar to mandir 
102    man1ext .. man9ext 
103        Extensions for manpage files. 
104    pkgdatadir 
105        The directory for installing idiosyncratic read-only 
106        architecture-independent data files for this program. 
107    pkgincludedir 
108        The directory for installing header files to be included by user programs 
109        with the C ``#include`` preprocessor directive. 
110    pkglibdir 
111        The directory for object files and libraries of object code. 
112    pkglibexecdir 
113        The directory for installing executable programs to be run by other 
114        programs rather than by users. 
115   
116  .. _GNU installation directories: http://www.gnu.org/prep/standards/html_node/Directory-Variables.html 
117  .. _GNU Coding Standards: http://www.gnu.org/prep/standards/html_node/
